# CoG
[Paper][ACL 2026] *CoG: Controllable Graph Reasoning via Relational Blueprints and Failure-Aware Refinement over Knowledge Graphs*

## Knowledge Graph and Datasets

Before running **CoG**, please first prepare the required knowledge graph and benchmark datasets.  
We use **CWQ**, **WebQSP**, and **GrailQA** for evaluation. The datasets should be placed under the `data/` directory.  
If aliases or additional processed files are needed, please organize them in the corresponding subdirectories.
